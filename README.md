# Excel Dashboard Project – Data Cleaning, Validation & Reporting

This project demonstrates how Excel can be used for data cleaning, validation, reconciliation, and dashboard reporting for business insights.

## Skills Demonstrated

- Data cleaning and preparation
- Data validation and reconciliation
- Use of Excel formulas 
- Dashboard creation for reporting and analysis
- Structured data organization for reporting workflows

## Files Included

- excel-sales-dashboard-data-quality.xlsx

## Dashboard Features

- Interactive dashboard built using Pivot Tables and charts
- Data quality checks and structured data preparation
- Summary insights for business reporting

This project reflects practical data management and reporting tasks similar to real-world business operations.
